# 📝 Text-as-Data Exercises

**👋 Welcome!**  
These exercises are designed for practice and exploration. The goal is to understand the methods, observe outputs, and think critically about the results — not to produce perfect code.  

**⚠️ Important:** If you do **not** save your own copy in Google Drive, any changes you make will be lost! 

If this is your **first Python experience**, try to do a little bit each day 🐢. Don’t try to solve everything at once.

---

## 📚 Exercise Structure

There are two Google Colab notebooks:

1. **Python Basics**  
   - Learn Python essentials: variables, lists, loops, functions, pandas.

2. **Text-as-Data Methods**  
   - Explore text analysis: word counts, document-feature matrices (DFM), dictionaries, TF-IDF, embeddings, topic modeling, sentiment analysis, and text generation.
