# text-as-data-exercises

This repository contains two Google Colab notebooks:

1. **Python Basics**  
   Introduction to Python, notebooks, and pandas.

2. **Textual Analysis with Python**  
   Toy examples covering word counts, dictionary analysis, TF-IDF, embeddings, topic modeling, sentiment analysis, and text generation.

## How to start

Open the first notebook in Google Colab:

[Open Python Basics in Colab](https://colab.research.google.com/github/YangliuF95/text-as-data-exercises/blob/main/Python_basics.ipynb)

Before you start, create your own copy:

1. Click **File**
2. Click **Save a copy in Drive**
3. Work on your own saved copy

Then complete the notebook from top to bottom.

## Then continue

After finishing the Python Basics notebook, open the second notebook for textual analysis:

[Open Textual Analysis with Python in Colab](https://colab.research.google.com/github/YangliuF95/text-as-data-exercises/blob/main/textual_analysis_with_python.ipynb)

Again, click **File > Save a copy in Drive** before you start.
